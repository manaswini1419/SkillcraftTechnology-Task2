# SkillCraft Technology – Task 2 (Stopwatch)
This repository contains the submission for Task 2 of the SkillCraft Technology internship.
## Project: Stopwatch Web Application
A simple and interactive stopwatch built using core web technologies.  
It allows users to start, pause, resume, and reset time accurately.
## Features
- Start the stopwatch
- Pause and resume functionality
- Reset the timer
- Real-time time update
- Clean and user-friendly interface
## Technologies Used
- HTML
- CSS
- JavaScript
## Description
This project demonstrates the use of JavaScript for time-based functions and DOM manipulation, along with HTML for structure and CSS for styling.
